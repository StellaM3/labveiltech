Pourquoi ce site, pourquoi Hugo ?#

Pour documenter ma veille technologique et stratégique, j’ai choisi de créer ce site avec Hugo, un générateur de site statique (SSG).

Pourquoi un SSG ?#

  • Performance : pages rapides à charger
  • Sécurité : pas de base de données à exposer
  • Simplicité : versionnée avec Git, hébergée sur GitHub Pages
  • Pérennité : le contenu m’appartient, je le gère en Markdown

Pourquoi Hugo ?#

  • Très rapide à générer
  • Flexible pour créer une architecture claire (articles, catégories, taxonomies)
  • Compatible avec de nombreux thèmes adaptés au blogging professionnel

C’est un choix aligné avec mes valeurs de maîtrise, autonomie et sobriété technique.
